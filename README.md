

Automated Video Cutter & Shorts Generator built with Python and FFmpeg.

AutoCut Clipper YouTube adalah tool berbasis Python yang digunakan untuk memotong video secara otomatis berdasarkan rentang waktu tertentu serta menghasilkan output dalam berbagai format (Landscape, Portrait, dan Shorts Style).

Project ini dirancang untuk mempercepat workflow content creation seperti YouTube Shorts, TikTok, Instagram Reels, dan highlight video.
